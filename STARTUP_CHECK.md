```
Cannot read properties of undefined (reading 'of')
```
**解决**: 已修复，应该不再出现

#### 2. content.js 错误
```
Cannot read properties of null (reading '2')
```
**解决**: 清理缓存后应该解决

#### 3. 500 错误
```
Failed to load resource: 500 (Internal Server Error)
```
**解决**: 组件已修复，应该正常加载

#### 4. 端口占用
```
Port 3000 is already in use
```
**解决**: 
```bash
lsof -ti:3000 | xargs kill -9
npm run dev
```
