## 🚀 如何运行测试

### 方法 1: Playwright (推荐)
```bash
# 安装 Playwright
npm run test:e2e

# 运行测试
npm run test:e2e -- --headed

# 快速测试
npx playwright test tests/e2e/quick-test.spec.ts --headed
```

### 方法 2: Puppeteer
