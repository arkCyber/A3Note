# A3Note 手动功能测试指南

## 测试准备

### 环境要求
- ✅ 浏览器: Chrome/Safari/Firefox (最新版本)
- ✅ 开发服务器: 运行在 http://localhost:1420
- ✅ 测试时间: 约 30-45 分钟

### 启动应用
```bash
# 浏览器模式（推荐用于快速测试）
npm run dev

# Tauri 桌面模式（完整功能测试）
npm run tauri:dev
```

## 附录

### 常见问题

**Q: 浏览器模式和 Tauri 模式有什么区别？**
A: 浏览器模式使用模拟数据，Tauri 模式访问真实文件系统。详见 `BROWSER_MODE.md`。

**Q: 如何重置应用状态？**
A: 清除浏览器 localStorage 或删除 Tauri 配置文件。

**Q: 测试数据会保存吗？**
A: 浏览器模式不保存，Tauri 模式保存到磁盘。

### 测试技巧

1. **使用浏览器开发工具**
   - Console: 查看日志和错误
   - Network: 检查请求
   - Performance: 分析性能

2. **快速重置**
   - 刷新页面重置浏览器模式
   - 重启应用重置 Tauri 模式

3. **并行测试**
   - 可以同时运行浏览器和 Tauri 模式
   - 对比两种模式的行为
