# Documentation Index

This directory contains project documentation organized by purpose.

## Structure

- `guides/`
  - End-user and developer setup guides
  - Testing and execution guides
- `development/`
  - Architecture, implementation, standards, and planning notes
- `reports/`
  - Audit reports, completion reports, plugin reports, and status summaries
