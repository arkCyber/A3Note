## 🚀 使用方法

### 打开插件市场

**方法 1**: 点击工具栏的 📦 图标

**方法 2**: 在代码中调用
```typescript
setPluginMarketplaceOpen(true);
```

### 搜索插件

1. 在搜索框输入关键词
2. 支持搜索：
   - 插件名称
   - 描述
   - 作者名

### 过滤插件

1. 点击分类标签（如 "productivity"）
2. 只显示该分类的插件

### 排序插件

选择排序方式：
- **Most Downloaded** - 按下载量排序
- **Highest Rated** - 按评分排序
- **Name (A-Z)** - 按名称排序

### 安装插件

1. 找到想要的插件
2. 点击 **Install** 按钮
3. 等待安装完成
4. 插件自动注册到系统

### 启用/禁用插件

1. 已安装的插件显示 **Enabled** 或 **Enable** 按钮
2. 点击切换启用状态
3. 绿色表示已启用

### 卸载插件

1. 点击 **Uninstall** 按钮
2. 确认卸载
3. 插件从系统移除

## 🔧 技术实现

### 插件下载流程

```
1. 用户点击 Install
   ↓
2. 解析 GitHub 仓库 URL
   ↓
3. 下载 manifest.json
   ↓
4. 下载 main.js
   ↓
5. 下载 styles.css (可选)
   ↓
6. 验证 manifest
   ↓
7. 保存到本地
   ↓
8. 注册到 PluginManager
   ↓
9. 完成安装
```

### GitHub 文件 URL 格式

```
https://raw.githubusercontent.com/{owner}/{repo}/{version}/manifest.json
https://raw.githubusercontent.com/{owner}/{repo}/{version}/main.js
https://raw.githubusercontent.com/{owner}/{repo}/{version}/styles.css
```

### 本地存储结构

```
.a3note/
  plugins/
    {plugin-id}/
      manifest.json
      main.js
      styles.css
      data.json
```
