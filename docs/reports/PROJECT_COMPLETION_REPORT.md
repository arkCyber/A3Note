## 💡 使用指南

### 用户操作

#### 切换语言

1. 点击工具栏右侧的"设置"图标
2. 在"外观"部分找到"语言"下拉框
3. 选择您想要的语言（24 种可选）
4. 界面立即切换

**推荐语言**（完整翻译）:
- 中文（简体）
- English
- 日本語
- 한국어

#### 使用功能

- **新建文件**: 点击工具栏"新建文件"按钮或按 ⌘+N
- **保存文件**: 点击工具栏"保存"按钮或按 ⌘+S
- **搜索**: 点击工具栏"搜索"按钮或按 ⌘+Shift+F
- **右键菜单**: 在侧边栏文件上右键点击
- **双击打开**: 在侧边栏双击文件打开

### 开发者扩展

#### 添加新语言

1. 复制 `src/i18n/locales/ko-KR/index.ts`
2. 重命名为目标语言（如 `fr-FR/index.ts`）
3. 翻译所有文本
4. 在 `src/i18n/index.ts` 中导入并添加
5. 运行测试验证

#### 添加新功能

1. 在相应组件中添加功能
2. 添加翻译键到所有语言文件
3. 编写测试
4. 更新文档
