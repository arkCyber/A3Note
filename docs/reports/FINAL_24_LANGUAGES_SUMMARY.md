## 🎯 使用指南

### 最终用户

#### 如何切换语言

1. **打开设置**
   - 点击工具栏右侧的齿轮图标

2. **选择语言**
   - 在"外观"部分找到"语言"下拉框
   - 点击下拉框查看所有 24 种语言

3. **切换语言**
   - 选择您想要的语言
   - 界面立即切换

4. **推荐语言**（完整翻译）
   - 中文（简体）
   - English
   - 日本語
   - 한국어

#### 语言持久化

- 您的语言选择会自动保存
- 下次打开应用时自动使用上次选择的语言
- 无需任何额外操作

### 开发者

#### 如何添加新语言

**示例：添加法语（fr-FR）**

1. **复制现有语言文件**
```bash
cp src/i18n/locales/ko-KR/index.ts src/i18n/locales/fr-FR/index.ts
```

2. **翻译所有文本**
```typescript
// src/i18n/locales/fr-FR/index.ts
export default {
  common: {
    save: 'Enregistrer',  // 翻译为法语
    cancel: 'Annuler',
    // ... 其他翻译
  },
  // ... 其他命名空间
};
```

3. **导入新语言**
```typescript
// src/i18n/index.ts
import frFR from './locales/fr-FR';

const resources = {
  'zh-CN': zhCN,
  'en-US': enUS,
  'ja-JP': jaJP,
  'ko-KR': koKR,
  'fr-FR': frFR,  // 添加法语
};
```

4. **测试**
```bash
npm test -- i18n.test.ts
```
