## 🎯 基本功能

### 1. 文件管理

#### 新建文件
- **方法 1**: 点击工具栏"新建文件"按钮
- **方法 2**: 按快捷键 `⌘+N`（Mac）或 `Ctrl+N`（Windows）
- **方法 3**: 在侧边栏右键 > 新建文件

#### 打开文件
- **方法 1**: 在侧边栏单击文件
- **方法 2**: 在侧边栏双击文件
- **方法 3**: 在侧边栏右键 > 打开

#### 保存文件
- **方法 1**: 点击工具栏"保存"按钮
- **方法 2**: 按快捷键 `⌘+S`（Mac）或 `Ctrl+S`（Windows）
- **自动保存**: 启用后自动保存（默认 2 秒延迟）

#### 删除文件
- **方法**: 在侧边栏右键文件 > 删除
- **确认**: 系统会要求确认删除操作

#### 重命名文件
- **方法**: 在侧边栏右键文件 > 重命名
- **输入**: 输入新文件名并确认

### 2. 文件夹管理

#### 新建文件夹
- **方法**: 在侧边栏右键文件夹 > 新建文件夹
- **输入**: 输入文件夹名称

#### 展开/折叠文件夹
- **方法**: 点击文件夹名称左侧的箭头图标

### 3. 编辑功能

#### Markdown 编辑
- 支持完整的 Markdown 语法
- 实时预览（如果启用）
- 语法高亮

#### 文本操作
- **复制**: `⌘+C` / `Ctrl+C`
- **粘贴**: `⌘+V` / `Ctrl+V`
- **剪切**: `⌘+X` / `Ctrl+X`
- **撤销**: `⌘+Z` / `Ctrl+Z`
- **重做**: `⌘+Shift+Z` / `Ctrl+Shift+Z`

### 4. 搜索功能

#### 打开搜索
- **方法 1**: 点击工具栏"搜索"按钮
- **方法 2**: 按快捷键 `⌘+Shift+F`

#### 搜索内容
- 输入搜索关键词
- 查看搜索结果
- 点击结果打开文件

### 5. 设置

#### 打开设置
- **方法**: 点击工具栏右侧的"设置"图标（齿轮）

#### 可配置项
- **外观**:
  - 主题（亮色/暗色）
  - 语言（24 种可选）
  - 字体大小

- **编辑器**:
  - 自动保存
  - 自动保存延迟
  - 拼写检查
  - 行号显示
  - 自动换行
  - 制表符大小

## 🌍 语言切换

### 支持的语言

#### 完整翻译（推荐）
1. **中文（简体）** - 默认语言
2. **English** - 英语
3. **日本語** - 日语
4. **한국어** - 韩语

#### 框架支持（回退到中文）
5. 中文（繁體）
6. ไทย（泰语）
7. Tiếng Việt（越南语）
8. Bahasa Indonesia（印尼语）
9. Bahasa Melayu（马来语）
10. हिन्दी（印地语）
11. Français（法语）
12. Deutsch（德语）
13. Español（西班牙语）
14. Italiano（意大利语）
15. Português（葡萄牙语）
16. Русский（俄语）
17. Polski（波兰语）
18. Nederlands（荷兰语）
19. Svenska（瑞典语）
20. Türkçe（土耳其语）
21. العربية（阿拉伯语）
22. עברית（希伯来语）
23. فارسی（波斯语）
24. Українська（乌克兰语）

### 如何切换语言

1. **打开设置**
   - 点击工具栏右侧的齿轮图标

2. **选择语言**
   - 在"外观"部分找到"语言"下拉框
   - 点击下拉框查看所有语言

3. **切换语言**
   - 选择您想要的语言
   - 界面立即切换（无需刷新）

4. **语言保存**
   - 您的选择会自动保存
   - 下次打开应用时自动使用

### 语言切换示例

```
中文界面：
- 设置 → 外观 → 语言 → 选择 "English"

English界面：
- Settings → Appearance → Language → Select "日本語"

日本語界面：
- 設定 → 外観 → 言語 → 選択 "한국어"

한국어界面：
- 설정 → 외관 → 언어 → 선택 "中文（简体）"
```

## ❓ 常见问题

### Q1: 如何更改主题？

**A**: 
1. 打开设置（点击齿轮图标）
2. 在"外观"部分找到"主题"
3. 选择"亮色"或"暗色（暖色调）"

### Q2: 语言切换后界面没有变化？

**A**: 
- 确保选择的是完整翻译的语言（中文、英语、日语、韩语）
- 其他语言会回退到中文显示
- 刷新页面试试

### Q3: 如何启用自动保存？

**A**: 
1. 打开设置
2. 在"编辑器"部分找到"自动保存"
3. 勾选复选框
4. 调整"自动保存延迟"（1-10 秒）

### Q4: 文件保存在哪里？

**A**: 
- A3Note 使用浏览器的 localStorage 存储
- 文件保存在浏览器本地
- 清除浏览器数据会删除文件

### Q5: 如何导出文件？

**A**: 
- 当前版本暂不支持导出
- 可以复制文本内容到其他应用

### Q6: 支持哪些 Markdown 语法？

**A**: 
- 标题（# ## ###）
- 列表（- * 1.）
- 粗体（**text**）
- 斜体（*text*）
- 代码（`code`）
- 代码块（```）
- 链接（[text](url)）
- 图片（![alt](url)）

### Q7: 如何报告问题？

**A**: 
- 查看浏览器控制台错误信息
- 记录操作步骤
- 联系开发团队

### Q8: 应用支持移动设备吗？

**A**: 
- 当前版本主要针对桌面浏览器
- 移动设备可以访问但体验可能不佳
- 未来版本会优化移动体验
