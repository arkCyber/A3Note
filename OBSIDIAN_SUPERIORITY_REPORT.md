```

- 查看文档结构

- 折叠/展开章节

```
点击顶部 "Template" 按钮 → 选择模板 → 填写变量 → 应用

- 选择内置模板
- 自定义变量
- 实时预览
- 一键插入

#### 4. 书签系统
```
点击顶部 ★ 按钮 → 查看书签 → 点击 + 添加当前文件
```
- 添加常用笔记
- 分类管理
- 快速访问
