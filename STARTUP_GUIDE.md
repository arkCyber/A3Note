## 🔧 重启开发服务器

### 方法 1: 软重启（推荐）

服务器已在运行，只需刷新浏览器：

```bash

Cmd+Shift+R  (Mac)
Ctrl+Shift+R (Windows/Linux)
```

### 方法 2: 完全重启

如果需要完全重启：

```bash
# 1. 停止当前服务器
# 在终端按 Ctrl+C

# 2. 清理缓存（可选）
rm -rf node_modules/.vite

# 3. 重新启动
npm run dev
```

## 🐛 问题排查

### 如果看到错误

#### 导入错误
```
错误: Cannot find module 'xxx'
解决: npm install
```

#### 图标不显示
```
错误: 图标位置空白
解决: 检查 lucide-react 是否安装
     npm install lucide-react
```

#### 热重载不工作
```
错误: 修改代码后不更新
解决: 
1. Ctrl+C 停止服务器
2. rm -rf node_modules/.vite
3. npm run dev
```

#### TypeScript 错误
```
错误: 类型错误
解决: npm run type-check
```
