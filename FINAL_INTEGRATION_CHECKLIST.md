## 🔧 常见问题排查

### 问题 1: 导入错误

**症状**: 控制台显示 "Cannot find module"

**解决**:
```bash
npm install
npm run dev
```

### 问题 2: 图标不显示

**症状**: 图标位置空白

**检查**:
- lucide-react 是否安装
- 图标名称是否正确
- 导入语句是否正确

**解决**:
```bash
npm install lucide-react
```

### 问题 3: 热重载不工作

**症状**: 修改代码后页面不更新

**解决**:
```bash
# 停止服务器
Ctrl+C

# 清理缓存
rm -rf node_modules/.vite

# 重新启动
npm run dev
```

### 问题 4: TypeScript 错误

**症状**: 类型错误提示

**检查**:
- 所有导入是否正确
- 类型定义是否完整
- tsconfig.json 配置

**解决**:
```bash
npm run type-check
```
