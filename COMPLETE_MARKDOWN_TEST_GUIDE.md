## 📝 完整测试清单

### 测试 1: 标题 (Headings)

**输入**:
```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

**预期效果**:
- `# 一级标题` → 2em 大小，粗体 700，亮色
- `## 二级标题` → 1.6em 大小，粗体 600
- `### 三级标题` → 1.3em 大小，粗体 600
- `#### 四级标题` → 1.1em 大小，粗体 600
- `##### 五级标题` → 1em 大小，粗体 600
- `###### 六级标题` → 0.9em 大小，粗体 600
- 所有 `#` 符号变小变淡

**验收标准**:
- ✅ 整行大小立即变化
- ✅ 文字加粗显示
- ✅ 不同层级大小不同
- ✅ `#` 符号淡化（50% 透明度）

## 🎯 问题排查

### 问题 1: 样式不显示

**检查**:
1. 浏览器是否硬刷新
2. CSS 文件是否加载
3. 开发者工具是否有错误
4. 元素是否有正确的 CSS 类

**解决**:
- 硬刷新浏览器
- 检查网络请求
- 查看控制台错误

### 问题 2: 标记不淡化

**检查**:
1. `.cm-formatting-mark` 类是否应用
2. CSS 透明度是否正确
3. 是否有其他样式覆盖

**解决**:
- 检查元素的 computed styles
- 确认 CSS 优先级
- 添加 `!important` 如果需要

### 问题 3: 性能问题

**检查**:
1. 文件大小
2. 装饰数量
3. 内存占用
4. CPU 使用率

**解决**:
- 优化装饰构建逻辑
- 只处理可见范围
- 使用增量更新
